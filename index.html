Treasure-Hunt Project for the module CO1111.

Embark on a thrilling Treasure Hunt like no other! Solve captivating puzzles, decode secret messages, and uncover hidden riches as you navigate this exciting web-based quest. Sharpen your wits and prepare to be amazed at every turn. Adventure awaits—are you ready to claim your prize?

How Does It Work?

Our Treasure Hunt relies on real-world data from the Code Cyprus Treasure Hunt API. This API provides questions one at a time, and you must be at the correct location to answer them properly. We also integrate with the Testing API so you can practice and refine your puzzle-solving skills without affecting real scores. Each correct answer gets you closer to victory—but beware of wrong answers and time limits!
